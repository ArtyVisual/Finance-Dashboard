# Finance Dashboard

A modern finance dashboard to track income, expenses, and financial insights.

## Features

- Dashboard overview (balance, income, expenses)
- Charts (income vs expenses, category breakdown)
- Transactions table with search & filtering
- Role-based UI (viewer/admin)
- Insights (savings rate, top categories, trends)
- Dark mode
- Responsive with mobile drawer navigation

## Tech Stack

- React + Vite
- Tailwind CSS + Custom CSS variables
- Recharts
- React Router

## Setup

```bash
npm install
npm run dev
